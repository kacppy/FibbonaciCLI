# FibbonaciCLI
Python project that shows the Fibonacci Sequence, depending on the arguments provided in the CLI.

# Usage

```
-h, --help: Display help message.
<value>: Specify a value for Fibonacci Sequence

Example:
python FibbonaciCLI.py 15
```
